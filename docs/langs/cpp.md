## **1. Variable Declaration**

**Syntax:**

```metal
<int|float|string|bool> <variable> = <value>
```

**Description:**

* Statically types variables like C++.
* Numbers, strings, and booleans supported.
* ChatGPT can simulate strict typing humorously.

**Examples:**

```metal
int HP = 100
float DAMAGE = 75.5
string HERO_NAME = "SKIBIDI"
bool IS_ALIVE = true
```

**Humor Tip:**
Use absurdly epic types like `int DRAGON_CRUSHER = 9001`.

## **3. Functions / Methods**

**Syntax:**

```metal

    // block
    return <value>
}
```

**Description:**

* C++-style function definitions with return type.
* ChatGPT simulates calling functions and returning results.

**Example:**

```metal

DRAGON_HP = ATTACK(HERO_NAME, DRAGON_HP)
```

